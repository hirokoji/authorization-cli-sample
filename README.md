# authorization-server-sample

## Install

### 1. Generate keys
```shell
ssh-keygen -t rsa -b 4096 -m PEM -f keys/jwtRS256.key
openssl rsa -in keys/jwtRS256.key -pubout -outform PEM -out keys/jwtRS256.key.pub
```
### 2. Install libraries
```shell
npm install
```
